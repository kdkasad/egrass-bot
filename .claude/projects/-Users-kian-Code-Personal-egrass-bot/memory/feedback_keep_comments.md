Do not remove existing comments when editing code.

**Why:** The user intentionally writes comments for clarity and wants the codebase to stay clean and readable.

**How to apply:** When modifying code, preserve all existing comments. Only remove a comment if the user explicitly asks for it or if the code it describes is being deleted.
